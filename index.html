jQuery CheckAll - Demo

Usage :
  • Include Jquery in the head section.
  • Include jquery.metalCheckAll plugin after jquery lib.
  • For full documentation, visit Github Documentation.
Example 1 - Using checkbox and select by class name.
Click Me 





jQuery
$('#checkAll_example1').metalCheckAll({
	target : ['.inputCheck']
});
Example 2 - Using checkbox and select by element name.
Click Me 





$('#checkAll_example2').metalCheckAll({
	target : ['input[name=example2]']
});
Example 3 - Using checkbox and select by ID name.
Click Me 





$('#checkAll_example3').metalCheckAll({
	target : ['#example3']
});
Example 4 - Using Button and select by class name. 





$('#checkAll_example4').metalCheckAll({
	target : ['.example4'],
	btnValueChecked : 'Click ME', // Button Text before check
	btnValueUnChecked : 'UnCheck Me' // Button text after check
});
Example 5 - Using Button and select by input type checkbox(this will find all checkbox on page). 





$('#checkAll_example5').metalCheckAll({
	target : ['input[type=checkbox]'],
	btnValueChecked : 'Click ME', // Button Text before check
	btnValueUnChecked : 'UnCheck Me' // Button text after check
});
Example 6 - Using Button and select by multiple selector. 





$('#checkAll_example6').metalCheckAll({
	target : ['input[name=a]','.example6','#example6','input[data-type="example6"]'], // arrays value
	btnValueChecked : 'Click ME', // Button Text before check
	btnValueUnChecked : 'UnCheck Me' // Button text after check
});
